Ordenando Texto

Um estudante de lingua estrangeira está aprendendo novas palavras para enriquecer o seu vocabulário, para tanto ele gosta de anotar as palavas das frases em ordem alfabética.

O desafio de agora é ordenar as palavras (em javascript) da Frase a seguir em ordem alfabética: {{ vm.texto }}

O Resultado deve ser igual a String abaixo

                  {{ vm.resultadoCorreto }}

Ao finalizar o seu desenvolvimento você deve inserir a sua string ordenada no atributo "vm.resultado" do controller. E será exibido abaixo:

                    {{ vm.resultado }}
